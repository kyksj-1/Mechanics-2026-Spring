### 题1: 虫口模型与 Logistic 映射

1. 对于迭代方程
    
    $$x_{n+1}=1-\mu x_{n}^{2}$$
    
    分析不同的 $\mu\in(0,2)$ 时 $x_{n}$ 的长时间行为，以及其与初值 $x_{0}\in(-1,1)$ 的关系。（2分）
    
2. 将上述方程换成
    
    $$x_{n+1}=\cos x_{n}-\mu x_{n}^{2}$$
    
    合理选择 $\mu,x_{0}$ 的区间，并分析 $x_{n}$ 的长时间行为。（1分）
    
3. 比较两种情况，你发现了什么？（1分）

### 题2: 开心消消乐

在一个 $L \times L$ 的网格（即每行每列均有 $L$ 个格子）上定义如下消消乐游戏：

1. 在最开始，格子上没有任何方块。
    
2. 每次挑选任意一个格子，放上去一个方块。每放上去一个方块，系统开始演化一个时间步。一个时间步内的演化规则见后面一条。
    
3. 如果一个格子上的方块数目大于等于4，则消去4个方块，并**记录一次消去**。这四个方块会移动并累加到上下左右的4个格点（它们各自方块数加1）。如果一个格子的两个邻居同时消去并传递方块给它，则它的方块数量加2，即影响可叠加。考虑开放边界条件：若边界格点消去，则有一个或两个方块移动出系统。每次同时消去所有当前高度大于等于4的格子，并**记录下总的消去次数**。直到场上没有任何格子的方块数大于等于4，这一个时间步结束。
    
4. 每一步的**得分 $s$** 即为这一步的总消去次数。
    

一个时间步的例子：

Plaintext

```
1 3 1      1 3 1      1 4 1      3 0 2      在初始网格[2,2]位置放置一个方块
3 3 1  ->  3 4 1  ->  4 0 2  ->  0 2 2      总共消去了3次“4”，所以这一步得分为3
2 2 3      2 2 3      2 3 3      3 3 3
```

1. 选取网格大小 $L=32$，演化足够多的时间步，并记录网格平均方块密度 $n$（$n=\frac{N}{L^{2}}$，其中 $N$ 是场上的方块总数）随演化时间 $t$ 的关系。你发现了什么？（2分）
    
2. 选取并固定一个合适的系统尺寸 $L$（取决于你的计算能力），统计长时间演化时每步得分的频率分布 $P(s)$。你发现了什么规律？（2分）
    
3. 选取多个不同的系统尺寸 $L$ 重复（2）的过程，验证你发现的统计规律。系统尺寸对这一规律有什么影响？（1分）
    
4. 你能解释你在（2）中发现的规律吗？有什么别的思考？（1分）
