### 5. 广义不确定性原理证明（3.5.1）

```

**第二步：施瓦茨不等式应用**
```
σ_A²σ_B² = ⟨f|f⟩⟨g|g⟩ ≥ |⟨f|g⟩|² ✓
```

**第三步：虚部分离（关键！）**
```
关键恒等式：|z|² = [Re(z)]² + [Im(z)]² ≥ [Im(z)]² = [1/(2i)(z - z*)]²
```

验证这个恒等式：
```
Im(z) = (z - z*)/(2i)
[Im(z)]² = [(z - z*)/(2i)]² = (z - z*)²/(−4)
但应该是虚部本身的平方：[Im(z)]² = |Im(z)|²

正确的恒等式是：
Im(z) = (z - z*)/(2i)
|Im(z)| = |z - z*|/(2)
[Im(z)]² = |z - z*|²/4 = [(z - z*)*(z* - z)]/(4) = |z - z*|²/4

实际上，对于任意复数 z：
|z|² ≥ [Im(z)]² 当且仅当 Re(z) = 0 时取等号
```

文档的处理（802行）：
```
σ_A²σ_B² ≥ [1/(2i)(⟨f|g⟩ - ⟨g|f⟩)]²
```

这里 1/(2i)(z - z*) 是虚部除以 i 的表达，符号正确 ✓

**关键计算：⟨f|g⟩ - ⟨g|f⟩**（806-814行）

验证：
```
⟨f|g⟩ = ⟨(Â-⟨A⟩)Ψ|(B̂-⟨B⟩)Ψ⟩
      = ⟨Ψ|(Â-⟨A⟩)(B̂-⟨B⟩)Ψ⟩ （利用厄米性）
      = ⟨ÂB̂⟩ - ⟨A⟩⟨B⟩ ✓
```

类似地：
```
⟨g|f⟩ = ⟨B̂Â⟩ - ⟨A⟩⟨B⟩
```

因此：
```
⟨f|g⟩ - ⟨g|f⟩ = ⟨ÂB̂ - B̂Â⟩ = ⟨[Â,B̂]⟩ ✓
```

**最终结论**：
```
σ_A²σ_B² ≥ [1/(2i)⟨[Â,B̂]⟩]² ✓
